## 🚀 Getting Started

### 1. Prerequisites
Ensure you have [Node.js](https://nodejs.org/) installed. This project uses [pnpm](https://pnpm.io/) for package management.

### 2. Installation
Clone the repository and install dependencies:
```bash
git clone https://github.com/devtallha/portfolio.git
cd portfolio
pnpm install
```

### 3. Development
Run the development server:
```bash
pnpm dev
```
Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

### 4. Build
To create an optimized production build:
```bash
pnpm build
```
